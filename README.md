# moduloapp

Pagina web simple para calcular el inverso multiplicativo de `a mod n` con explicacion paso a paso.

## Ejecutar local

Abre `index.html` en el navegador.

## Deploy en GitHub Pages

Este repo ya incluye workflow en `.github/workflows/pages.yml`.

Al subirlo a GitHub:

1. Ve a `Settings > Pages`.
2. En `Build and deployment`, selecciona `Source: GitHub Actions`.
3. Haz push a `main` y el workflow publicara la pagina.
